== Description ==

The M77 Spotify Embed plugin makes it super easy to embed songs, albums and playlists from Spotify directly into your posts and pages.

You will see a handly little meta box in your admin where you simply paste the Spotify URI. There's nothing more to it. If you'd like to change theme or customize your Spotify Player, don't worry. With a few clicks in the settings panel, you've updated all your previous posts.

== Installation ==

Install and activate from within Wordpress or download and extract the zip file in the wp-content/plugins/ directory of your WordPress installation, then activate the Plugin from Plugins page.
